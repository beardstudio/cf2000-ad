Instructions SQL pour les bases de données

Il existe plusieurs commandes pour manipuler les bases de données, par exemple :

  • SELECT nom, commentaire FROM messages ORDER BY date DESC
  • INSERT INTO messages(nom, commentaire, genre, date) VALUES('Marie', 'Bonjour,...', 1, CURRENT_TIMESTAMP)
  • UPDATE messages SET email = 'marie.dufour@cf2m.be' WHERE id=4
  • DELETE FROM messages WHERE id=5
